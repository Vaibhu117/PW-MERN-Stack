Absolute:1.Syntax{position:absolute;}2.With relative positioning, an element can be moved up, down, left, or right from its normal position.3.It will not break the normal document flow to position the element on the page.4.The element will leave the space at its original position & other content will not be adjusted to fit into any gap left by the element.Relative:1.Syntax{position:relative;}2.With absolute positioning, an element is positioned relative to its nearest positioned ancestor.3.It will break the normal document flow to position the element on the page.4.The element will not leave any space in its original position.
